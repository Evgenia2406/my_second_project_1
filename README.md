Интерактивная Канбан-доска для визуализации рабочих процессов и управления задачами по методологии Agile. Приложение помогает отслеживать прогресс проекта от идеи до реализации.

## ✨ Основные возможности
* **Управление задачами**: создание, редактирование и удаление карточек.
* **Drag-and-Drop**: интуитивное перемещение задач между колонками.
